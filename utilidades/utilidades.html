  • Ayuda

    Al dar clic en esta utilidad, aparecerá un corto formulario para registrar sus inquietudes o dudas administrativas. El Asesor de servicio al estudiante dará respuesta lo antes posible.

  • Foro

    Es un espacio de discusión que le permitirá interactuar con otros estudiantes y el tutor.

  • Actividades

    En este espacio encuentra un conjunto de exámenes de entrenamiento sobre cada módulo. Estas actividades tienen calificación, si no las realiza, no podrá avanzar al siguiente módulo.

  • Calificaciones

    En este espacio usted podrá revisar sus calificaciones módulo a módulo, de esta manera sabrá que le hace falta para seguir avanzando.

  • Glosario

    En este ícono encontrará una lista de palabras específicas del curso, con la explicación breve de cada una.

  • Juegos

    En este espacio encontrará un conjunto de juegos de entrenamiento sobre este módulo. Estas actividades no tendrán calificación, pero le permitirán autoevaluar su conocimiento.

  • Para llevar a casa

    En este espacio podrá descargar información relevante para su uso.
